Library Management Application
This is study project allows users create base of students (users of library) and control which student take some book.

Now user of this app can:
- add, edit, delete students
- add, edit, delete books (description of books - title, author).

Later will add some features like:
- store information, which books takes some student (and when student must return this books).
- inform manager of library, which student didn`t return books in time.
